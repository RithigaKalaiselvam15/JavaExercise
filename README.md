Employee Management System
(Collections + OOP + Constructors + Java 8)-
Problem Statement : Create an Employee Management System.
Requirements - Create an Employee class: - id, name, department,  salary .
Use: Constructor chaining , Getters & setters,Store employees in a List<Employee> .
Operations: Add employee, Remove employee by ID , Display all employees , Find employees with salary > X (use Streams) , Sort employees by salary (ascending) .
Concepts Used - OOP , Constructors, Collections (List), Java 8 Streams ,Control statements .
