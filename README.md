##  Problem Statement

Create a simple **Employee Management System** in Java.

---

##  Requirements

### Employee Class

Create an `Employee` class with the following attributes:

- `id`
- `name`
- `department`
- `salary`

---

### Class Design

The `Employee` class should use:

- Constructor chaining
- Getters and setters
- Proper encapsulation (OOP principles)

---

### Employee Storage

- Store employees in a `List<Employee>`

---

##  Operations

The system should support the following operations:

- **Add employee**
- **Remove employee by ID**
- **Display all employees**
- **Find employees with salary greater than X**  
  *(Using Java 8 Streams)*
- **Sort employees by salary (ascending order)**

---

##  Concepts Used

-  Object-Oriented Programming (OOP)
-  Constructors & Constructor Chaining
-  Java Collections (`List`)
-  Java 8 Streams
-  Control Statements
